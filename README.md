## 앞으로 프로젝트 진행 방식

#### 앞으로 프로젝트 진행은 fork 떠서 작성 및 자신의 branch 에 풀리퀘 날리기

- 해당 프로젝트로 직접적으로 브랜치만 다르게 하니 풀리퀘의 기록이 작성이 안됨.
- 따라서 해당 프로젝트를 fork 한 후 개인 계정의 repo 에서 작업을 한 뒤, 자신의 branch 로 풀리퀘를 날리는 방식을 사용한다.
